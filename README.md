# ربات تلگرام قیمت ارز، طلا و ارز دیجیتال

ربات تلگرامی که قیمت لحظه‌ای دلار، یورو، پوند، طلا، بیت‌کوین، اتریوم و تتر را
از tgju.org اسکرپ کرده و در اختیار کاربران (چت خصوصی یا گروه) قرار می‌دهد.

## ویژگی‌های این نسخه

1. **اسکرپ پس‌زمینه‌ای هر ۳۰ ثانیه:** به‌جای اسکرپ در لحظه‌ی هر درخواست، یک
   ترد پس‌زمینه هر ۳۰ ثانیه یک‌بار قیمت‌ها را اسکرپ کرده و در فایل
   `data/prices.json` ذخیره می‌کند. هندلرهای ربات فقط همین فایل را می‌خوانند،
   یعنی پاسخ به کاربر آنی است و فشار کمتری هم روی tgju.org وارد می‌شود.
2. **پنل مدیریتی:** با دستور `/admin` (فقط برای آیدی‌های داخل `ADMIN_IDS`)
   می‌توانید لیست کاربران، لیست گروه‌های مجاز/در انتظار تایید و آمار کلی
   ربات را ببینید و گروه‌ها را تایید یا لغو مجوز کنید.
3. **دارایی‌های بیشتر:** علاوه بر دلار، یورو و طلا، حالا پوند انگلیس،
   بیت‌کوین، اتریوم و تتر هم اضافه شده‌اند (`config.py` → `ASSETS`).
4. **ماژول `keep_alive.py` دست‌نخورده مانده** تا دیپلوی روی Render طبق روال
   قبلی، سرویس را زنده نگه دارد.

## راه‌اندازی

```bash
pip install -r requirements.txt
```

فایل `.env` را باز کنید و مقداردهی کنید:

```
BOT_TOKEN=توکن-ربات-شما
ADMIN_IDS=آیدی-عددی-شما
```

برای گرفتن آیدی عددی‌تان: ربات را اجرا کنید، در چت خصوصی دستور `/myid` را
بزنید، عدد نمایش داده‌شده را داخل `ADMIN_IDS` قرار دهید و ربات را دوباره اجرا
کنید. اگر چند نفر باید دسترسی ادمین داشته باشند، آیدی‌ها را با کاما جدا کنید.

اجرا:

```bash
python bot.py
```

## نحوه‌ی کار پنل مدیریت

- در چت خصوصی با ربات، دستور `/admin` را بزنید.
- **لیست کاربران:** همه‌ی کسانی که تاکنون با ربات تعامل داشته‌اند (آیدی +
  نام کاربری).
- **گروه‌های در انتظار تایید:** وقتی ربات را به یک گروه اضافه می‌کنید، آن
  گروه به‌طور خودکار (با اولین استفاده از یک دستور مثل `/start`) در این
  لیست ثبت می‌شود اما تا وقتی از پنل تایید نکنید، ربات در آن گروه پاسخ
  نمی‌دهد (به‌جز خود شما به‌عنوان ادمین). برای پیدا کردن آیدی گروه هم
  می‌توانید داخل گروه دستور `/myid` را بزنید.
- **گروه‌های مجاز:** لیست گروه‌هایی که تایید شده‌اند؛ از همین‌جا می‌توانید
  مجوز را لغو کنید.
- **آمار کلی:** تعداد کاربران، گروه‌های مجاز و در انتظار.

> نکته مهم درباره‌ی هاست رایگان Render: فایل‌های data/*.json (قیمت‌ها،
> کاربران، گروه‌ها) روی دیسک سرویس ذخیره می‌شوند. در پلن رایگان Render که
> Persistent Disk ندارد، این فایل‌ها بعد از هر دیپلوی/ری‌استارت پاک می‌شوند
> (یعنی باید دوباره گروه‌ها را تایید کنید). اگر می‌خواهید این اطلاعات دائمی
> بماند، از یک Persistent Disk در تنظیمات Render استفاده کنید و مسیر پوشه‌ی
> پروژه (یا حداقل data/) را روی آن دیسک قرار دهید.

## افزودن دارایی جدید

برای اضافه کردن هر ارز/دارایی دیگر از tgju.org، کافیست یک آیتم جدید به
دیکشنری ASSETS در config.py اضافه کنید (کلیدواژه‌ی متنی که در جدول
tgju.org ظاهر می‌شود + یک عدد minimum برای فیلتر کردن مقادیر نادرست). بقیه‌ی
بخش‌ها (کیبورد، دستورات، تشخیص کلیدواژه در چت) به‌صورت خودکار آپدیت می‌شوند.

> از آنجا که دسترسی زنده به tgju.org برای تست نبود، مقادیر minimum برای
> دارایی‌های جدید (پوند، بیت‌کوین، اتریوم، تتر) بر پایه‌ی قیمت‌های تقریبی
> امروز تنظیم شده‌اند. بعد از اولین اجرا روی سرور، حتماً خروجی /btc، /eth،
> /usdt و /gbp را چک کنید؛ اگر پیام «یافت نشد» دیدید، به احتمال زیاد
> کلیدواژه یا minimum آن دارایی نیاز به تنظیم دقیق‌تر دارد.

## دستورات ربات

| دستور | توضیح |
|---|---|
| /start | شروع و نمایش کیبورد اصلی |
| /all یا /market | نمایش همه‌ی قیمت‌ها |
| /usd /eur /gbp /gold /btc /eth /usdt | قیمت هر دارایی |
| /myid | دریافت آیدی عددی شما و گروه |
| /admin | پنل مدیریت (فقط برای ADMIN_IDS) |
